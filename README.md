# Точка контакта — статичный лендинг

Сайт работает как статичная страница без бэкенда.

## Что внутри

- Основной файл: `index.html`
- Контакты для участия: ссылки на VK, Telegram и Email
- Плавный скролл к разделу программы
- Адаптивная верстка для мобильных устройств

## Запуск

Откройте `index.html` в браузере или опубликуйте репозиторий через GitHub Pages.
