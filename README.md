# Точка контакта — форма с надежной доставкой

## Что реализовано

- Отдельный клиентский файл формы: `form-handler.js`
- API обработчик: `POST /api/feedback`
- Сохранение заявок в SQLite базу `leads.db`
- Уведомления в 3 канала:
  - email на несколько адресов
  - Telegram в несколько чатов
  - SMS на несколько номеров (Twilio)
- Получение списка заявок: `GET /api/leads`

## Быстрый старт

1. Скопируйте `.env.example` в `.env` и заполните значения.
2. Установите зависимости:

   ```bash
   npm install
   ```

3. Запустите сервер:

   ```bash
   npm start
   ```

4. Откройте сайт: `http://localhost:3000`

## Проверка

- Health endpoint: `GET /health`
- Отправка формы: заполните поля на главной странице и нажмите "Отправить заявку"
- Просмотр заявок:
  - без токена: если `ADMIN_TOKEN` не задан
  - с токеном: передайте `x-admin-token: <ADMIN_TOKEN>`
  - endpoint: `GET /api/leads?limit=50`
